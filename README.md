# 乞丐生存游戏 - 更新说明

## 最新更新内容

### 配方系统界面调整

1. **移除行动区域的配方按钮**
   - 从行动系统定义中删除了配方按钮
   - 行动区域现在只显示：乞讨、搜寻、休息

2. **在卡牌组合区域添加配方按钮**
   - 在组合按钮旁边新增了"配方"按钮
   - 使用flex布局使两个按钮并排显示
   - 配方按钮点击后打开配方窗口

3. **改进配方窗口显示**
   - 已解锁的配方显示完整信息（名称、材料、描述）
   - 未解锁的配方显示为问号（???），隐藏具体信息
   - 增加了配方解锁的成就感

### 游戏机制优化

- 保持了原有的配方解锁机制（通过制作或在贫民窟学习）
- 配方窗口支持点击任意位置关闭
- 未解锁的配方会在满足条件后自动显示详细信息

## 游戏玩法说明

### 基本操作
- 将资源拖放到组合区域进行制作
- 点击"组合"按钮尝试制作物品
- 点击"配方"按钮查看已解锁的配方
- 探索不同地点获取资源
- 管理状态（饥饿、寒冷、健康、精神、声望）

### 配方系统
- 初始时没有解锁任何配方
- 通过制作新物品或在贫民窟学习可以解锁配方
- 未解锁的配方显示为问号，增加探索乐趣
- 已解锁的配方可以在配方窗口中查看详细信息

### 制作配方
- 泥碗：1破碗 + 2泥巴
- 简易衣物：3破布
- 拐杖：1木棍 + 2破布
- 篝火：2木棍 + 1燧石
